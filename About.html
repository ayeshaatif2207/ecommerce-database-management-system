About Stupify

Stupify is a database management interface designed for e-commerce store operations. Built as a simplified Shopify-style database system, Stupify enables store managers and staff to efficiently manage products, customers, orders, and inventory through an intuitive web interface connected to a robust MySQL relational database.

The platform demonstrates enterprise-grade database design with role-based access control, allowing different user types (Admin, Staff, Analyst) to interact with data according to their responsibilities. All transactions are managed through a Python Flask backend that connects to a normalized MySQL database adhering to third normal form (3NF) principles.

Database Schema

The database consists of 9 tables modelling a complete Shopify-style e-commerce system:

  • CUSTOMER — customer profiles (name, phone, email)
  • ADDRESS — customer shipping and billing addresses
  • POSTALCODE — city and country lookup by postal code
  • PRODUCT — products with pricing, stock levels, and category associations
  • CATEGORY — product categories and classifications
  • CART & CART ITEM — shopping cart sessions and line items
  • ORDER & ORDER ITEM — placed orders and associated line items
  • PAYMENT — payment records linked to orders
  • SHIPPING METHOD — available shipping options and costs
  • REVIEW — customer product reviews and ratings

Features

  • Role-Based Access Control — Three distinct user roles (Admin, Manager, Analyst) with granular MySQL-enforced privileges
  • Product Management — View, search, filter, and add products with real-time inventory tracking
  • Normalized Database Design — All tables conform to Third Normal Form (3NF) eliminating data redundancy
  • Transaction Support — ACID-compliant transactions ensuring data consistency across operations
  • Clean Web Interface — Dark-themed, responsive design built with HTML, CSS, and Flask
  • Real-Time Queries — Live data retrieval from MySQL with JOINs and aggregations

Tech Stack

  • HTML5 & CSS3 — semantic markup and modern styling
  • Python 3 with Flask — lightweight web framework for server-side routing
  • Jinja2 — templating engine for dynamic content rendering
  • MySQL 8.x — relational database with native user privilege management
  • mysql-connector-python — Python MySQL database driver
